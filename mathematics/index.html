      0About this site

      This site contains weekly problem sets for the tutorial classes and may contain notes and examples from the lectures. Material will be added to the site continuously throughout the semester, so initially you may find it to be a bit empty. To navigate the site and see the material, click the navigation button at the top left — the three vertical bars (burger icon).
      Slides from the lectures can be found at the course site in Brightspace (login required).
      The official course description can be found in the university's course catalogue.
